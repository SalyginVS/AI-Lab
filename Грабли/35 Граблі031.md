# Грабля #31 — scp не создаёт промежуточные директории

## Симптом

`scp` файлов в `~/llm-gateway/gateway/__init__.py` возвращает:
```
dest open "llm-gateway/gateway/__init__.py": No such file or directory
```

## Причина

scp не создаёт промежуточные директории. Директория `gateway/` не существовала на сервере.

## Решение

```bash
ssh vladimir@192.168.0.128 "mkdir -p ~/llm-gateway/gateway"
```

Затем повторить scp-команды.

## Урок

При деплое нового пакета (директории) через scp — всегда `mkdir -p` перед массовым копированием. Включать эту команду в инструкции деплоя.
