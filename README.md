## Полный пакет из 5 документов. Вместе с существующими Паспортом v33 и Архитектурой v1.19 они образуют полный комплект для переноса проекта.

Структура пакета:

00. Portability Gap Analysis — аналитический документ: 16 категорий данных из 30+ файлов этапов, которые не вошли в паспорт и архитектуру. Приоритизация по критичности для переноса.
   
1.  Операционный справочник — полная операционная инструкция: управление сервисами, токены, модели, логи, orchestrator (CLI + HTTP), RAG/Text-to-SQL, сеть и безопасность, benchmarks, обновление Ollama, восстановление с нуля, диагностика 5 типичных проблем.
   
2.  Gateway техническая спецификация — внутренняя реализация: 11 модулей, 7 реализационных паттернов (стриминг httpx, tool_calls конвертация, multimodal, reasoning policy, OOM, auth, self-call), Pydantic модели с полными диапазонами, structured logging формат, pipeline registry.
   
3.  Конфигурация клиента — полная клиентская конфигурация: config.yaml Continue (9 моделей, 12 context providers, 6 prompts), 3 MCP сервера (YAML configs), rules 3 уровней, Copilot BYOK (settings.json + расширение), 17 грабель Continue с решениями, диагностика.
   
4.  Enterprise Transfer Brief — стратегический документ для ONDO: 7 паттернов переноса (инвариант vs изменяемое), 9 governance principles, 5 CIO экспериментов (статус + следующие шаги), полный реестр файлов сервера и клиента, вендорский benchmark-фреймворк.
