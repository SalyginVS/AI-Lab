# Грабли 37 — SyntaxError: global declaration после использования переменной

**Симптом:** `SyntaxError: name 'NUM_CTX' is used prior to global declaration` при запуске text2sql_poc.py.

**Причина:** В Python `global NUM_CTX` должен быть в начале функции, до любого использования переменной (включая чтение через `args.num_ctx`).

**Решение:** Перенести `global NUM_CTX` в первую строку `main()`, до парсинга аргументов.

**Урок:** Python-линтер не ловит это до runtime. В скриптах с глобальным состоянием (`global`) проверять позицию объявления.
