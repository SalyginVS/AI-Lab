# glm-4.7-flash

Альтернативная agentic coding модель. MoE-архитектура: 30B параметров, ~3B активных. MIT License.

| Параметр | Значение |
|----------|----------|
| Размер | 19 ГБ |
| Архитектура | glm4moelite (MoE) |
| Параметры | 29.9B (3B активных) |
| Контекст | 202752 (максимум), 8192 (наш дефолт) |
| Capabilities | completion, tools, thinking |
| Лицензия | MIT |

## Роли в платформе
- **Agent mode** — альтернативная (tools работают)
- **Agentic coding** — создание файлов, запуск в терминале
- **Chat / Edit / Apply** — доступен

## Особенности
- Tool calling совместим с `convert_ollama_tool_calls_to_openai()` без патчей
- MoE: быстрее на inference чем dense 30B при сопоставимом качестве
- Огромный контекст (200K+), но в нашем стеке ограничен 8192 для защиты от OOM
