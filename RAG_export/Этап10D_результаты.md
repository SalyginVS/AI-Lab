## Замечание: DeprecationWarning в orchestrator.py

При запуске orchestrator обнаружен `DeprecationWarning: datetime.datetime.utcnow()` — Python 3.12+ считает `datetime.utcnow()` устаревшим. Правильная замена: `datetime.now(datetime.UTC)`.

**Статус:** не блокирующее, зафиксировано как технический долг (Грабли #34). Исправление — при следующем касании orchestrator.py.

## Грабли

### Грабли #34: datetime.utcnow() DeprecationWarning в orchestrator.py

**Симптом:** При запуске orchestrator.py на Python 3.12+ в stderr выводится `DeprecationWarning: datetime.datetime.utcnow() is deprecated`.

**Причина:** Python 3.12 deprecated `datetime.utcnow()` в пользу timezone-aware альтернатив.

**Fix:** Заменить `datetime.utcnow()` на `datetime.now(datetime.UTC)` в orchestrator.py. Не блокирующее — warning не влияет на функциональность.
