# Грабля #30 — systemd ExecStart placeholder (status=203/EXEC)

## Симптом

После обновления `llm-gateway.service` сервис не стартует: `status=203/EXEC`.

## Причина

В ExecStart оказался placeholder `/.../venv/bin/python run.py` вместо реального пути `/home/vladimir/llm-gateway/venv/bin/python run.py`.

## Решение

```bash
sudo sed -i 's|ExecStart=/.../venv/bin/python run.py|ExecStart=/home/vladimir/llm-gateway/venv/bin/python run.py|' /etc/systemd/system/llm-gateway.service
sudo systemctl daemon-reload
sudo systemctl restart llm-gateway
```

## Урок

Всегда давать абсолютные пути в systemd unit-файлах. Инструкции деплоя должны содержать реальные пути, не placeholder'ы.
