# Этап 14B — Gateway Auth + Audit

**Gateway v0.10.0 → v0.11.0.** Mandatory per-user Bearer auth + user_id audit trail.

## Что сделано

- Новый модуль `auth.py` — per-user token validation, mandatory (sys.exit без токенов)
- ENV-based token storage: `LLM_GATEWAY_TOKENS` в `.env` (chmod 600), `EnvironmentFile=` в systemd
- 2 пользователя: `vladimir`, `roadwarrior`
- `user_id` в каждом structured log event (8 точек log_data: 5 chat + 3 embed)
- `/health`: `auth_enabled=true`, `user_count=2`
- RAG MCP: debug prints удалены (3 штуки)
- test_stage14b.sh: **14/14 PASS**

## Regression

RAG MCP → gateway `/v1/embeddings` → 401. Cascading: mandatory auth сломал server-to-server вызов. Fix: Bearer auth в `gateway_embeddings.py` + EnvironmentFile в `rag-mcp.service`.
