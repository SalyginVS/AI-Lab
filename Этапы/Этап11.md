# Этап 011 — MCP: RAG / Docs Search

## Цель

Создать custom MCP-сервер для семантического поиска по документации лаборатории через ChromaDB + qwen3-embedding.

## Результат

Custom RAG MCP server работает end-to-end. 133 файла / 286 chunks проиндексированы. 3 tools доступны в Continue Agent mode. Retrieval: 9/10 релевантных, кириллица работает, cross-tool (git+RAG) подтверждён.

## Стек

- FastMCP (mcp 1.27.0), streamable-http, порт 8100
- ChromaDB 1.5.5 PersistentClient, collection `lab_docs`
- qwen3-embedding (4096d) через gateway /v1/embeddings
- systemd `rag-mcp.service`

## Ключевые решения

- [[ADR-014 streamable-http для серверных MCP]] — не STDIO (Windows-only), не SSE (deprecated)
- ChromaDB embedded для PoC, миграция на Qdrant при >100K docs
