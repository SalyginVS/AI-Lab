# Грабли #34: datetime.utcnow() DeprecationWarning в orchestrator.py

## Симптом

При запуске `orchestrator.py` на Python 3.12+ в stderr выводится:

```
DeprecationWarning: datetime.datetime.utcnow() is deprecated and scheduled for removal in a future version. Use timezone-aware objects to represent datetimes in UTC: datetime.datetime.now(datetime.UTC)
```

## Причина

Python 3.12 deprecated `datetime.utcnow()` — метод возвращает naive datetime без timezone info, что создаёт потенциальные баги при работе с часовыми поясами.

## Fix

Заменить в `orchestrator.py`:

```python
# Было:
from datetime import datetime
timestamp = datetime.utcnow().strftime(...)

# Стало:
from datetime import datetime, UTC
timestamp = datetime.now(UTC).strftime(...)
```

## Статус

Не блокирующее — warning не влияет на функциональность. Исправить при следующем касании orchestrator.py.
